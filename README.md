# 🧹 Data Analytics - Week 1 (Data Cleaning Project)

## 🚀 Overview

This repository contains my **Week 1 Data Analytics Project** completed as part of training at DecodeLabs.

In this week, I focused on:

* **Data Cleaning & Preparation**
* Converting raw data into a clean and reliable dataset

---

## 📁 Files Included

* `Cleaned_Dataset.xlsx` → Final cleaned dataset
* `Data Analytics Week-1.ipynb` → Python notebook with cleaning steps

---

## 🎯 Objective

The main goal of this project is to:

* Clean raw dataset
* Handle missing values
* Remove duplicates
* Fix incorrect data formats

---

## 🛠 Tools & Technologies

* Python (Pandas)
* Jupyter Notebook
* Microsoft Excel

---

## 🧹 Data Cleaning Steps Performed

### 🔹 Handling Missing Values

* Identified null values in dataset
* Filled missing values (e.g., CouponCode)

### 🔹 Removing Duplicates

* Removed duplicate rows
* Ensured no duplicate Order IDs

### 🔹 Fixing Data Formats

* Converted date column to proper format
* Removed unnecessary time from date
* Ensured correct numeric data types

### 🔹 Data Validation

* Verified:

  * No missing values
  * No duplicate records
  * Correct date format

---

## 📊 Final Result

* Dataset is clean and ready for analysis
* No missing values
* No duplicate records
* Consistent and correct data format
